# WB Analytics Dashboard

Дашборд аналитики продаж Wildberries с ИИ-анализом.

## Возможности

- **Командный центр**: ключевые метрики, хиты и аутсайдеры продаж, ИИ-анализ
- **Матрица решений (BCG)**: классификация товаров по выручке и динамике
- **Дашборд действий**: рекомендации по ценам, рекламе, запасам, выводу товаров

## Запуск локально

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск бэкенда
cd backend
uvicorn main:app --reload --port 8000

# Открыть dashboard.html в браузере
```

## Деплой на Railway

1. Форкните репозиторий
2. Подключите к Railway
3. Добавьте переменную окружения `ANTHROPIC_API_KEY`
4. Deploy!

## Переменные окружения

- `ANTHROPIC_API_KEY` - API ключ Claude для ИИ-анализа (опционально)
