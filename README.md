# Omnilytic

Аналитический дашборд для продавцов на Wildberries. Загружайте данные через Excel или напрямую из WB API и получайте KPI, воронку продаж, BCG-матрицу товаров и рекомендации с ИИ-анализом.

## Возможности

- ✅ **Командный центр** — ключевые показатели, воронка продаж, хиты и зоны внимания *(работает)*
- ✅ **Юнит-экономика и тарифы** — детализация по товару за период, изменение тарифов *(работает)*. Требует WB API ключ и колонку «Артикул WB» (nmId) в Excel — ID из дропдауна автоматически резолвится в реальный nmId
- ⚠️ **Матрица решений** — группировка товаров по динамике роста и выручке (BCG) *(в разработке)*
- ⚠️ **План действий** — рекомендации по остаткам, падению продаж и точкам роста *(в разработке)*
- ⚠️ **ИИ-анализ** — автоматическая аналитика через Claude API *(в разработке)*
- ✅ **Загрузка данных** — Excel-файлы или напрямую из Wildberries Seller API *(работает)*

## Локальный запуск

```bash
pip install -r requirements.txt

cd backend
uvicorn main:app --reload --port 8000
```

После запуска дашборд доступен по адресу `http://127.0.0.1:8000`.

## Переменные окружения

- `ANTHROPIC_API_KEY` — API-ключ Claude для ИИ-анализа (опционально)
